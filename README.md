Videojuego de plataformas desarrollado con Phaser 3.

Acumular 100 puntos recolectando figuras geométricas antes de que se acabe el tiempo. Cuidado con las figuras rojas, te restan puntos.

Controles:
-Flecha izquierda / derecha: mover al personaje
-Espacio: saltar

Figuras:
| Figura    | Puntos |
| Cuadrado  |  +10   |
| Triángulo |  +15   |
| Rombo     |  +20   |
| Mala      |  -25   |

Tecnologías:
- [Phaser 3](https://phaser.io/)
- JavaScript
- Vite
